1. **`merge_check.py`** *(new file at project root)*
   - CLI script accepting `--ticket ISS-NNN`
   - Calls `python ticket.py view ISS-NNN` via `subprocess.run` (stdout captured)
   - Parses the `notes:` line of that output with regex `r"Branch\s+(feature/[^\s|]+)"` to extract branch name

### Implementation approach

`merge_check.py` design (module-level functions, testable without running main):

```
_get_ticket_notes(ticket_id) → str
    subprocess.run([sys.executable, "ticket.py", "view", ticket_id], capture_output=True, text=True)
    parse "notes: " line from stdout
    exit 2 on failure

_extract_branch(notes: str) → str
    re.search(r"Branch\s+(feature/[^\s|]+)", notes)
    exit 2 if no match

_is_merged(branch: str) → bool
    subprocess.run(["git", "branch", "--merged", "main"], capture_output=True, text=True)
    strip "* " from each line, check membership

check(ticket_id: str) → None
    calls _get_ticket_notes → _extract_branch → _is_merged
    prints result, sys.exit(0 or 1)

main() → argparse + check()
```
